# learn-git

A hands-on repository for learning and practicing core Git operations: **clone**, **pull**, **push**, and general repository management.

## Purpose

This repository is designed for beginners who want to get comfortable with Git by making real changes to Markdown files. Each contribution helps you practice the full Git workflow.

## Getting Started

1. **Fork** this repository to your own GitHub account.
2. **Clone** your fork to your local machine (see [GIT_GUIDE.md](GIT_GUIDE.md) for detailed instructions).
3. Create or edit a Markdown file.
4. **Commit** and **push** your changes back to GitHub.
5. Open a **Pull Request** to share your work.

## Contributing

- Add a new `.md` file with anything you've learned about Git.
- Edit an existing file to fix a typo or add more detail.
- Keep changes small and focused so they are easy to review.
